# SHIL Smart Solar Design - نسخه نهایی اصلاح‌شده

## نصب و اجرا

```bash
npm install
npm run dev
```

آدرس اجرا:

```text
http://localhost:5173/
```

## بیلد نهایی

```bash
npm run build
npm run preview
```

## اصلاحات این نسخه

- اصلاح منطق ضریب همزمانی بر اساس روش محاسبات:
  - لیست تجهیزات: ضریب برای هر تجهیز
  - توان کل و جریان کل: ضریب کلی
  - انرژی روزانه: بدون اعمال مجدد ضریب همزمانی
- اصلاح ضریب فصل کارکرد: فقط روی انرژی مصرفی اعمال می‌شود.
- افزودن امکان ثبت تجهیز جدید خارج از بانک در لیست تجهیزات و ذخیره به عنوان کاربرساخت.
- افزودن امکان ثبت پنل، باتری و اینورتر جدید به بانک اختصاصی کاربر.
- افزودن وضعیت کاربرساخت / در انتظار تایید ادمین برای تجهیزات سفارشی.
- اضافه شدن منطق تجهیزات موتوری سافت‌استارت / اینورتر داخلی با ضریب راه‌اندازی پیشنهادی 1.2.
- اصلاح چکیده اطلاعات:
  - جداسازی پنل از MPPT
  - انتقال MPPT به بلوک اینورتر
  - توضیح سری/موازی باتری
  - جداسازی نصب و فضا از حفاظت
  - افزودن حفاظت DC/AC و کابل‌ها
- اصلاح گزارش نهایی A4 برای چاپ و PDF با پس‌زمینه سفید و خوانایی در چاپ سیاه‌وسفید.
- اعمال تم ظاهری نئون/مهندسی برای فرم‌های ورودی محاسباتی.
