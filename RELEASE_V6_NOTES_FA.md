# نسخه V6 یکپارچه SHIL Solar

این نسخه بر پایه V5 ساخته شده و فازهای اصلی در یک خروجی واحد جمع شده‌اند:

- فلو کامل پروژه جدید با حالت Focus/Wizard
- مسیر PV: آفگرید، آنگرید، هیبرید
- مسیر برق اضطراری مستقل با اینورتر و باتری
- روش‌های محاسبه: جریان، توان، لیست تجهیزات، انرژی، پروفایل مصرف
- پروفایل مصرف با چهار بازه صبح/ظهر/عصر/شب و اعمال جمع انرژی در محاسبات
- شرایط محیطی با شهر پیش‌فرض اصفهان، جستجوی شهر، ضرایب تابش، دما، سایه، گردوغبار، زاویه و ارتفاع
- بانک تجهیزات مصرفی ۲۵۰ آیتمی، بانک پنل، اینورتر و باتری به صورت باز/بسته
- کنترل همخوانی تجهیزات، باتری، اینورتر، Surge موتور و ضریب همزمانی
- هشدار توان بالا در آفگرید و پیشنهاد اینورتر خارج از بانک یا پارالل هیبرید
- چکیده اطلاعات و خروجی نهایی A4 با PDF/PNG/Print
- سناریوهای آماده سبک/متوسط/سنگین با مسیر شرایط محیطی قبل از تجهیزات
- ارتباط با ما با تصویر برند و لینک SHIL.IR
- پنل ادمین برای مشاهده/اصلاح/حذف/انتقال پروژه به سناریو
- بهبود تم و کارت‌های هوشمند برای شروع مسیر طراحی

## اجرا

```bash
npm install
npm run dev
```
