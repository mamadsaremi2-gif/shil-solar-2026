# گزارش اصلاح نسخه موبایل SHIL

این نسخه برای تست موبایل واقعی اصلاح شد:

- اضافه شدن safe-area برای آیفون.
- جلوگیری از overflow افقی در همه صفحات.
- اصلاح داشبورد موبایل به حالت 2x2 برای کارت‌های آماری.
- کاهش ارتفاع و فضای مرده Hero و Header در موبایل.
- اصلاح صفحه پروژه جدید: کارت‌های مراحل در موبایل دو ستونه، خوانا و بدون بریدگی متن.
- اصلاح کارت‌های مسیر طراحی برای عدم overlap متن و آیکن.
- فشرده‌سازی spacing موبایل برای کاهش اسکرول اضافی.
- اصلاح Action Cardهای داشبورد در موبایل.
- فعال‌سازی route هوش مصنوعی SHIL از داشبورد.
- باقی ماندن AI بدون کلید API در کد؛ برای GitHub/Vercel امن است.
- Build تولیدی با موفقیت انجام شد.

نکته: برای فعال‌کردن AI در محیط production، کلید API نباید داخل کد قرار بگیرد؛ باید از Environment Variable یا backend proxy استفاده شود.
