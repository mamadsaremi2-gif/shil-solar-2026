# V13 Mobile App-like Engineering UX

این نسخه لایه نمایشی موبایل را به حالت اپ نصبی نزدیک می‌کند:

- هر مسیر موبایل در قالب صفحه‌ای به اندازه Viewport نمایش داده می‌شود.
- داشبورد موبایل فشرده‌تر شده تا اطلاعات اصلی بدون اسکرول سنگین دیده شود.
- صفحات پروژه دارای Header ثابت و Action Bar پایین صفحه هستند.
- محتوای هر مرحله داخل کارت اصلی اسکرول داخلی کنترل‌شده دارد.
- اسکرول افقی در کل اپ حذف شده است.
- کارت‌ها، فرم‌ها و چکیده‌ها در موبایل به شکل فشرده و مرحله‌ای نمایش داده می‌شوند.
- موتور محاسبات تغییر نکرده و فقط Presentation Layer موبایل بهینه شده است.
