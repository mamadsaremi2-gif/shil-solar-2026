# گزارش آپدیت حذف Legacy مسیر خورشیدی

این آپدیت روی نسخه `SHIL_src_contextual_engine_update` اعمال شد و هدف آن جلوگیری از تزریق خروجی‌های قدیمی در مسیر طراحی خورشیدی بود.

## موارد اعمال‌شده

### 1. موتور مرکزی خورشیدی
- خروجی خورشیدی از `src/engineering/solar/solarDesignEngine.js` مرجع اصلی شد.
- مسیر `runEngineeringPipeline` برای خورشیدی دیگر به Gateway قدیمی متکی نیست.
- در مرحله اجرای محاسبات، اگر `shil:solarSystemDesign` تأیید شده وجود داشته باشد، خروجی نهایی از همان طراحی تأییدشده خوانده می‌شود.

### 2. محاسبه انرژی بر اساس شرایط محیطی
- برای مسیرهای توان کل و جریان کل، انرژی مبنا از توان خام و ساعت آفتابی مؤثر شهر محاسبه می‌شود.
- برای نمونه اصفهان با `20kW` و `PSH=5.7` انرژی پایه `114kWh/day` می‌شود.
- ضریب طراحی روی توان و انرژی اعمال می‌شود.

### 3. انتخاب اینورتر
- اگر سناریو آفگرید/هیبرید باشد، انتخاب قبلی Utility یا On-Grid دیگر به‌عنوان انتخاب معتبر پذیرفته نمی‌شود.
- برای توان‌های بالاتر از 10kW، سیستم بین اینورترهای قابل پارالل زیر 10kW انتخاب می‌کند.
- برای نمونه `24kW`، پیشنهاد به سمت `3 × 8kW` می‌رود، نه `Utility 30kW`.

### 4. بانک پنل و آرایش PV
- تعداد پنل بر پایه انرژی طراحی، PSH، راندمان و توان پنل محاسبه می‌شود.
- آرایش استرینگ به تعداد اینورتر و MPPT وابسته شد.
- ممکن است برای رسیدن به استرینگ سالم چند پنل اضافه شود و این مورد هشدار داده می‌شود.

### 5. خروجی نهایی
- صفحه اجرای محاسبات برای خورشیدی ابتدا طراحی تأییدشده صفحه تنظیمات سیستم را می‌خواند.
- مقادیر اینورتر، پنل، باتری، MPPT و خودکفایی از خروجی مرکزی گرفته می‌شوند.

### 6. اعداد انگلیسی
- تبدیل اعداد فارسی/عربی به انگلیسی در بخش‌های خروجی اصلاح شد.
- `toFaDigits` عملاً خروجی فارسی تولید نمی‌کند و فقط متن ایمن‌شده را برمی‌گرداند.

## مواردی که هنوز کامل نشده‌اند

- طراحی حفاظت DC/AC هنوز کاملاً دیتاشیت‌محور نیست.
- انتخاب کابل هنوز نیازمند موتور افت ولتاژ، طول مسیر و شرایط نصب است.
- خروجی PDF همچنان نیازمند Fine Tune برای خلاصه یک‌صفحه‌ای نهایی است.
- مسیرهای Utility و Emergency باید بعد از تست مسیر Solar با همین الگو بازبینی شوند.

سناریو تست:

- PSH: 5.7
- راندمان: 92%
- توان خام: 20000W

- نوع سیستم: آفگرید یا هیبرید
- پنل پیش‌فرض: 620W
