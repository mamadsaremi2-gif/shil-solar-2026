# نصب سریع SHIL SOLAR Mobile V4

## پیش‌نیاز
- Node.js نسخه 18 یا 20
- npm

## اجرا روی سیستم شخصی
```bash
npm install
npm run dev
```

بعد از اجرا این آدرس را باز کنید:
```text
http://localhost:5173/
```

## اجرا روی موبایل داخل همان وای‌فای
```bash
npm run dev:host
```

بعد IP سیستم را در موبایل باز کنید، مثل:
```text
http://192.168.1.10:5173/
```

## ساخت نسخه نهایی برای هاست
```bash
npm run build
npm run preview
```

خروجی نهایی داخل پوشه `dist` ساخته می‌شود.

## نصب روی گوشی به صورت PWA
1. سایت را در Chrome موبایل باز کنید.
2. منوی سه‌نقطه را بزنید.
3. گزینه `Add to Home screen` یا `Install app` را انتخاب کنید.

## نکته کش مرورگر
اگر نسخه قدیمی نمایش داده شد، یک بار Hard Refresh بزنید یا کش سایت را پاک کنید.
