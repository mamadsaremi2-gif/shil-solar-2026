# SHIL SOLAR Mobile Standalone Release

این نسخه برای موبایل جدا شده است اما موتور محاسبات، دیتابیس، Supabase، بانک تجهیزات و سناریوها با نسخه اصلی مشترک می‌مانند.

## هدف
- هر صفحه در موبایل شبیه یک Workspace مستقل باشد.
- اسکرول افقی حذف شود.
- کارت‌ها و فرم‌ها فشرده شوند.
- نوار مرحله‌ها compact و قابل اسکرول افقی کنترل‌شده باشد.
- دکمه‌های مرحله در Bottom Action Bar قرار بگیرند.
- گزارش نهایی در موبایل به کارت‌های قابل خواندن تبدیل شود.

## فایل اصلی تغییر

- ایمپورت در `src/main.jsx`
