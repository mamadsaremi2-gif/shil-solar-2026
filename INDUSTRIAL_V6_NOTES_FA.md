# SHIL SOLAR - Industrial V6 Notes

این نسخه روی مرحله صنعتی کامل تمرکز دارد:

- تحلیل قابلیت اجرای صنعتی با امتیاز 0 تا 100
- محاسبه پوشش بکاپ نسبت به ساعت پشتیبانی موردنیاز مشتری
- محاسبه جریان DC دائم و لحظه‌ای برای کنترل غیرواقعی شدن کابل، فیوز و بانک باتری
- پیشنهاد ولتاژ DC مناسب بر اساس سطح توان
- تحلیل پوشش انرژی PV نسبت به مصرف روزانه
- نمایش کمبود/مازاد انرژی PV
- پیشنهاد اقدام اصلاحی در خروجی مهندسی و PDF
- تقویت Advisor برای خطاهای بحرانی جریان DC، بکاپ ناکافی و پوشش PV ناکافی

نکته: فایل .env.local داخل ZIP قرار داده نشده و باید فایل قبلی Supabase در پوشه اصلی پروژه نگه داشته شود.
