# بازسازی استاندارد موبایل SHIL - نسخه V28

در این نسخه، CSSهای تکراری و متناقض قبلی از مسیر اجرای اصلی حذف شدند و فقط یک فایل جامع برای قوانین موبایل فعال است:

```txt
src/styles/shil-mobile-standard-v28.css
```

فایل اصلی ورود اپ هم ساده شد:

```txt
src/main.jsx
```

## قوانین اعمال‌شده

- عرض کل اپ موبایل: `100vw` به‌صورت Fluid و بدون container ثابت اجباری.
- اسکرول عمودی: همیشه فعال برای بدنه و نواحی محتوا.
- اسکرول افقی: به‌صورت سراسری غیرفعال.
- اسکرول افقی فقط برای بلوک‌های مجاز مثل مسیر طراحی/Stepbar و بلوک‌های خاص افقی فعال است.
- همه کارت‌ها، آیکون‌ها، تیترها، دیتاها، فرم‌ها و جدول‌ها وسط‌چین شدند.
- جدول‌ها روی موبایل تنگ به‌صورت block/stack نمایش داده می‌شوند تا overflow افقی ایجاد نشود.
- Header/Footer با safe-area موبایل هماهنگ شد.
- سایز آیکون‌ها با متغیر مرکزی کنترل می‌شود: `--shil-icon`.
- فاصله‌ها، padding، gap، font-size و radius با متغیرهای مرکزی کنترل می‌شوند.
- کدهای فعال تکراری نسخه‌های قبلی از import اصلی حذف شدند.

## تست‌ها

- `npm run build` موفق شد.
- `npm run test:engineering` موفق شد.
