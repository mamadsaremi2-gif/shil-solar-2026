# نسخه V7 - Professional Solar Glass UI

این نسخه روی طراحی حرفه‌ای و یکپارچه کل اپ تمرکز دارد.

## تغییرات اصلی

- تم جدید Solar Glass + Tech با گرادیانت عمیق، کارت‌های شیشه‌ای و Glow خورشیدی/الکتریکی
- صفحه داشبورد حرفه‌ای‌تر با لوگوی مرکزی، کارت عنوان، Hero و دکمه‌های 3D
- استایل یکپارچه برای Wizard، Stepper، کارت‌ها، دکمه‌ها و فرم‌ها
- بهبود صفحه سناریوهای آماده با دسته‌بندی حرفه‌ای و مسیر کم‌حجم‌تر
- بهبود صفحه ارتباط با ما با قاب شیشه‌ای، لینک واضح SHIL.IR و کارت‌های اطلاعات تماس
- بهبود خروجی و کارت‌های وضعیت، هشدار و تحلیل
- حفظ فلو و منطق نسخه V6
