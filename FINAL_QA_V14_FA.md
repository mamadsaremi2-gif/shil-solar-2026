# SHIL SOLAR V14 Final QA

این نسخه با هدف یکپارچه‌سازی رنگ، مینیمال‌سازی کارت‌ها، حذف شلوغی داشبورد، تفکیک پروژه‌های کاربر از مدیریت، اصلاح کارتابل نظرات و کاهش اسکرول/فاصله‌های غیرضروری آماده شده است.

## موارد اعمال‌شده

- طراحی روشن‌تر و مدرن‌تر در کل اپ
- یکپارچه‌سازی کارت‌ها، فیلدها، متن‌ها و جدول‌ها
- Compact layout سراسری
- داشبورد با Hero اصلاح‌شده، عنوان خوانا و کارت‌های همسان
- حذف کارت وضعیت کاربر و تبدیل وضعیت شبکه به نشانگر کوچک آنلاین/آفلاین
- تغییر پروژه‌ها به «پروژه‌های من» و تفکیک در حال انجام / نهایی
- ایزوله‌سازی پروژه‌های هر کاربر در داشبورد عمومی
- باقی ماندن پروژه‌های همه کاربران در پنل مدیریت
- اصلاح صفحه آموزش و اعلام نظر کاربران با سربرگ هماهنگ
- متن جدید صفحه نظرات
- کارتابل نظرات شخصی کاربر
- حذف نظر فقط برای کاربر و باقی ماندن نسخه مدیریتی
- Density rule و Accordion-ready styling برای صفحات سنگین
- حذف/کاهش استیکرهای سنگین مسیر طراحی و جایگزینی با نئون آبی آرام
- بهینه‌سازی موبایل و حذف اسکرول افقی

## تست‌ها

- npm run build
- npm run test:engineering:all
