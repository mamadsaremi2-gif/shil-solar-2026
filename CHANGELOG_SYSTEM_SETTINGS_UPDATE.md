# SHIL System Settings Update

تغییرات اعمال‌شده در صفحه تنظیمات سیستم:

- بلوک کنترل طراحی حفظ شد و انتخاب‌های Off-Grid / On-Grid / Hybrid به فرمان ادامه مسیر و بانک‌ها متصل شدند.
- بلوک جدید «دیتاها و پارامترهای وارد شده مرحله قبل» اضافه شد.
- خروجی‌های مرحله قبل طبق روش انتخابی کاربر در این بلوک نمایش داده می‌شوند.
- روزهای خودکفایی و ضریب اطمینان استاندارد به داده‌های مرحله قبل اضافه شدند.
- بلوک «محاسبات با راندمان‌ها» اضافه شد تا توان مبنا، ضریب اطمینان، توان نهایی طراحی، راندمان/تلفات و منبع تابش را نشان دهد.
- بلوک «نتایج کلی و بانک‌های پیشنهادی» اضافه شد.
- انتخاب اینورتر پیشنهادی، باتری پیشنهادی و پنل پیشنهادی به بانک‌های SHIL متصل شد.
- انتخاب نوع اجرا روی پیشنهاد بانک اینورتر، نیاز باتری و بانک پنل اثر می‌گذارد.
