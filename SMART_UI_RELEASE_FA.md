# نسخه Smart Solar Intelligence UI

این نسخه داشبورد اصلی اپ را مطابق کانسپت Dark / Smart / Industrial بازطراحی می‌کند.

## تغییرات اصلی

- داشبورد شیشه‌ای و هوشمند مشابه نمونه تصویری
- Sidebar دسکتاپ با Active state، Badge و AI item
- KPI Cards برای تولید لحظه‌ای، تولید روزانه، صرفه‌جویی و وضعیت سیستم
- نقشه نیروگاه‌ها با Pin و Tooltip
- نمودار تولید انرژی SVG بدون وابستگی جدید
- Solar Digital Twin نمایشی
- پنل هشدارها
- پنل دستیار هوشمند AI
- Bottom Navigation برای موبایل
- Responsive کامل برای دسکتاپ، تبلت و موبایل

## فایل‌های تغییر کرده

- `src/pages/DashboardPage.jsx`
- `src/index.css`
