# نسخه نهایی اصلاح داشبورد موبایل SHIL

این نسخه با حفظ ساختار فعلی اپ اصلاح شده است.

## تغییرات اصلی

- تمرکز کامل روی موبایل و PWA
- حذف اسکرول افقی در سطح `html`, `body`, `#root` و داشبورد
- تبدیل کارت‌های بزرگ داشبورد به دکمه‌های خطی فشرده
- الهام از کلیدهای کیبورد کامپیوتری با افکت برجستگی و فشردن
- حفظ اسکرول عمودی
- مینیمال شدن متن‌ها و فیلدها
- کوچک‌سازی Hero داشبورد
- فشرده‌سازی بخش پروژه‌ها
- حفظ ساختار فعلی، Routeها و منطق اپ

## تست انجام‌شده

- `npm run build` با موفقیت اجرا شد.
- `npm run test:engineering:all` با موفقیت اجرا شد.
