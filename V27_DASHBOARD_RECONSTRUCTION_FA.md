# V27 Dashboard Reconstruction

این نسخه روی V26 پایدار ساخته شد و تمرکز آن بازطراحی داشبورد بعد از تثبیت Layout Engine است.

## تغییرات
- کاهش ارتفاع Hero داشبورد و کنترل Crop تصویر
- حذف Pill تکراری وضعیت شبکه داخل داشبورد
- تبدیل کارت‌های داشبورد به Grid فشرده سه‌ستونه موبایلی
- اصلاح تایپوگرافی عنوان داشبورد
- فشرده‌سازی کارت اطلاعات اصلی و Stats
- ایجاد حس Banking / Industrial / iOS26 بدون شکستن Shell پایدار V26
- حفظ Header/Footer و Scroll Engine نسخه V26

## نکته
این نسخه فقط داشبورد و لایه ظاهری مرتبط را بازسازی کرده و منطق محاسبات و مسیرهای پروژه دست‌نخورده مانده‌اند.
