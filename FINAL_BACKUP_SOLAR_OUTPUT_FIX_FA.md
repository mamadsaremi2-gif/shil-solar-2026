# اصلاحات نهایی مسیر برق اضطراری و خروجی موبایل

این نسخه شامل اصلاحات زیر است:

- تفکیک مسیر برق اضطراری از مسیر سولار در تنظیمات سیستم، چکیده اطلاعات و خروجی اجرایی.
- حذف نمایش آیتم‌های PV، MPPT، پنل، تابش، کابل PV و تولید خورشیدی در مسیر برق اضطراری.
- اضافه شدن منطق UPS/Backup در گزارش برق اضطراری: توان مصرف‌کننده، مدت زمان برق اضطراری، UPS/سانورتر، باتری، کابل باتری و حفاظت AC.
- حفظ ساختار گزارش خروجی در نسخه موبایل شبیه نسخه دسکتاپ/کتابچه‌ای، با Scale مناسب برای مطالعه و زوم.
- اصلاح صفحه چکیده و تنظیمات برای نمایش فیلدهای مخصوص برق اضطراری.
- سناریوهای آماده ابتدا وارد شرایط محیطی می‌شوند.
- دکمه خروج به خروج واقعی و Reload متصل است.
- Build پروژه قبل از خروجی گرفتن با موفقیت تست شد.

نکته: برای فعال‌سازی AI در محیط Production از Environment Variable امن استفاده شود و کلید API داخل کد قرار نگیرد.
