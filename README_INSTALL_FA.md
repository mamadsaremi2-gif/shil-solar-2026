# نسخه اصلاح‌شده نهایی SHIL SOLAR

## تغییرات انجام‌شده

- اتصال اپ به ساختار اصلی کامل شامل Dashboard، Workspace، Output، Contact، Scenarios، Equipment و Admin.
- اضافه شدن دکمه «+ پروژه جدید» به نوار بالای داشبورد.
- حذف دکمه‌های شناور وسط تصویر داشبورد.
- اصلاح صفحه «ارتباط با ما»:
  - جایگزینی تصویر اصلی با بنر جدید محصولات SHIL.
  - اضافه شدن لوگو، بازگشت و داشبورد روی بالای تصویر.
  - فقط ناحیه مشخص روی تصویر لینک مستقیم به SHIL.IR دارد؛ کل تصویر لینک نیست.
  - جایگزینی متن «مهندس صارمی» با «کارشناس فنی».
- اصلاح ورود:
  - ورود آفلاین به اپ اضافه شد.
  - ادمین بعد از ورود خودکار به پنل مدیریت منتقل نمی‌شود.
- اصلاح PWA:
  - آیکون‌های درست نصب موبایل تنظیم شد.
  - حالت standalone، portrait، RTL و cache آفلاین تنظیم شد.
  - اپ بعد از نصب قابلیت اجرا در حالت آفلاین دارد.
- همگام‌سازی:
  - پروژه‌ها ابتدا در localStorage ذخیره می‌شوند.
  - هنگام اتصال اینترنت و ورود کاربر، همگام‌سازی با سرور/Supabase انجام می‌شود.

## نصب و اجرا روی سیستم

```bash
npm install
npm run dev
```

## تست build

```bash
npm run build
```

## اجرای نسخه build شده

```bash
npm run preview
```

## ارسال به GitHub و بروزرسانی Vercel

```bash
git add .
git commit -m "final shil app fixes"
git push
```

بعد از push، Vercel به صورت خودکار build و deploy می‌کند.

## نصب روی موبایل

در Chrome یا Edge:

1. سایت را باز کنید.
2. منوی مرورگر را باز کنید.
3. گزینه Add to Home Screen یا Install App را بزنید.
4. برنامه روی موبایل نصب می‌شود و بعد از اولین بارگذاری، قابلیت اجرای آفلاین دارد.

## نکته مهم

فایل `npm run build` در همین نسخه تست شد و موفق بود.
