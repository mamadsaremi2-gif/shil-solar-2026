# گزارش اصلاح نسخه آماده تحویل SHIL Solar

## اصلاحات انجام‌شده

1. **اصلاح اسکریپت‌های Build در `package.json`**
   - `npm run build` از حالت وابسته به مسیر داخلی `node_modules/vite/bin/vite.js` به دستور استاندارد `vite build` تغییر کرد.
   - اسکریپت GitHub Pages نیز به `VITE_BASE=./ vite build` اصلاح شد.

2. **اضافه شدن تست مهندسی مستقل**
   - فایل `scripts/engineering-smoke-test.mjs` اضافه شد.
   - اسکریپت جدید `npm run test:engineering` در `package.json` اضافه شد.
   - این تست، موتور محاسبات را در دو سناریوی کلیدی بررسی می‌کند:
     - سیستم off-grid با بار روشنایی و پمپ موتوری
     - سیستم backup بدون پنل خورشیدی

3. **اصلاح Importهای حساس در فایل‌های JavaScript**
   - Importهای داخلی در فایل‌های `.js` که ممکن بود در اجرای مستقیم Node مشکل ایجاد کنند، با پسوند `.js` کامل شدند.
   - این موضوع برای تست‌های مستقل موتور محاسبات و اجرای خارج از Vite مهم است.

4. **شفاف‌سازی وضعیت Build**
   - در ZIP اولیه، پوشه `dist` وجود نداشت. در این نسخه، مستندات اصلاح شده‌اند تا مشخص باشد خروجی production باید با `npm ci && npm run build` ساخته شود.

## تست انجام‌شده

دستور زیر اجرا و با موفقیت کامل شد:

```bash
node scripts/engineering-smoke-test.mjs
```

خروجی:

```text
Engineering smoke tests passed
```

## محدودیت باقی‌مانده

در محیط بررسی، نصب کامل وابستگی‌ها با `npm ci` به دلیل timeout کامل نشد؛ بنابراین build نهایی Vite در همین محیط تأیید نشد. با این حال ساختار package اصلاح شده و تست مستقیم موتور محاسبات پاس شده است.

## دستورهای پیشنهادی برای تحویل نهایی روی سیستم توسعه‌دهنده

```bash
npm ci
npm run test:engineering
npm run build
npm run preview
```

اگر هر چهار دستور بدون خطا اجرا شدند، پروژه از نظر اجرایی آماده انتشار است.
