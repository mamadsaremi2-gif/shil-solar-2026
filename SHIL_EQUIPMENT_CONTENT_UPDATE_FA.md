# گزارش تغییرات محتوایی بانک تجهیزات SHIL

## تغییرات انجام‌شده

- ستون/فیلد «نوع پنل» از بانک تجهیزات، فرم افزودن تجهیز و نمایش خروجی حذف شد.
- بانک پنل خورشیدی به محصولات SHIL محدود و به‌روزرسانی شد:
  - SHIL 550W
  - SHIL 620W
  - SHIL 700W
- باتری‌های لیتیوم SHIL اضافه شدند:
  - 12.6V 100Ah
  - 25V 100Ah
  - 25V 200Ah
  - 51V 100Ah
  - 51V 200Ah
  - 51V 300Ah
- اینورترهای SHIL اضافه شدند:
  - آفگرید: 1.6kW/12V، 3.2kW/24V، 4kW/24V، 6kW/48V، 8kW/48V، 10kW/48V
  - هیبرید: 6kW/48V، 8kW/48V، 11kW/48V
- دسته «شارژ کنترلر» از کتابخانه تجهیزات حذف شد.
- انتخاب اینورتر در فرم پروژه بر اساس نوع سیستم فیلتر می‌شود:
  - در آفگرید و بکاپ فقط اینورترهای آفگرید نمایش داده می‌شوند.
  - در هیبرید فقط اینورترهای هیبرید نمایش داده می‌شوند.
- متن‌های مربوط به شارژ کنترلر مستقل در خروجی به MPPT داخلی تغییر داده شد تا با حذف شارژ کنترلر از بانک تجهیزات سازگار باشد.
- سناریوهای آماده از فیلد نوع پنل پاک شدند تا به تکنولوژی ثابت وابسته نباشند.

## نتیجه تست

دستور زیر با موفقیت اجرا شد:

```bash
npm run test:engineering:all
```

خروجی:

```text
Engineering smoke tests passed
Real engineering calculation tests passed
Engineering validation tests passed
All engineering test suites passed
```

## نکته اجرایی

در این نسخه پارامترهای محاسباتی داخلی مثل ضریب دمایی پنل و محدوده MPPT همچنان برای صحت طراحی مهندسی باقی مانده‌اند، اما «نوع پنل» به کاربر نمایش داده نمی‌شود و در بانک تجهیزات ثبت نشده است.
