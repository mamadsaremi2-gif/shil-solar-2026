# تنظیم کارت‌های صفحه انتخاب مسیر پروژه

دو کارت صفحه «انتخاب مسیر پروژه» از فایل زیر خوانده می‌شوند:

```txt
public/project-path-cards.json
```

برای تغییر عنوان، توضیح یا تصویر کارت‌ها فقط همین فایل را ویرایش کنید. تعداد کارت‌ها در UI حداکثر دو عدد در نظر گرفته شده است.

نمونه هر کارت:

```json
{
  "key": "solar",
  "title": "اجرای پروژه با پنل خورشیدی",
  "description": "طراحی سیستم خورشیدی با پنل، باتری، اینورتر و حفاظت",
  "image": "/assets/shil/execution/solar-execution.svg",
  "calculationDomain": "solar"
}
```

پس از انتخاب هر کارت و زدن دکمه تأیید، مسیر پروژه ذخیره می‌شود و کاربر وارد صفحه انتخاب روش محاسبات می‌شود:

```txt
/new-project/method
```
