# SHIL Summary + AI Preview Update

این به‌روزرسانی صفحه چکیده اطلاعات را بعد از تأیید پیکربندی تنظیمات کامل می‌کند.

## تغییرات
- صفحه چکیده به‌صورت بلوک‌بندی بازنویسی شد.
- خلاصه اطلاعات اصلی هر مرحله شامل پروژه، شرایط محیطی، مصرف، تجهیزات و پیکربندی سیستم نمایش داده می‌شود.
- بلوک اختیاری «هوش مصنوعی SHIL» اضافه شد.
- اگر عکس محل نصب در شرایط محیطی وجود داشته باشد، دکمه AI درخواست ساخت تصویر مجازی محل نصب با پنل خورشیدی را آماده می‌کند.
- تعداد واقعی پنل، اینورتر، باتری و توان اجرایی محاسبه‌شده در همان بلوک نمایش داده می‌شود.
- تأیید چکیده کاربر را به صفحه اجرای محاسبات منتقل می‌کند.
