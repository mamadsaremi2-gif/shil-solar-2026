# SHIL Before-Test Update

این نسخه بر اساس ۹ نکته اعلام‌شده آماده شده است:

1. افزودن جستجوی بالای صفحه سناریوهای آماده خورشیدی و برق اضطراری.
2. حفظ مسیر سناریو: انتخاب سناریو → شرایط محیطی → لیست تجهیزات همان شاخه → موتور محاسبات یکپارچه.
3. تفکیک خروجی محاسبات به اینورتر، پنل، باتری، MPPT، حفاظت، کابل و فضای نصب.
4. پشتیبانی بهتر از تایپ فارسی/انگلیسی با `dir=auto` و `unicode-bidi: plaintext`.
5. روشن‌تر شدن بکگراند اصلی و خروج از حس تاریک/بی‌روح.
6. پایین‌تر آمدن کارت لاگین تا لوگوی پس‌زمینه گم نشود.
7. تبدیل بلوک‌ها، هدرها و فیلدها به گلاسه روشن با متن سیاه و بولد.
8. اصلاح متن‌های خراب فارسی صفحه لاگین.
9. بازگردانی حس آیکون‌های گلاسه با نئون آبی/بنفش.

دستور تست:

```bash
npm install
npm run engine:smoke
npm run dev -- --force
```
