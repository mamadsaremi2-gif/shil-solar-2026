# SHIL Ready Scenario Environment + Equipment Flow Update

این آپدیت مسیر سناریوهای آماده خورشیدی را اصلاح می‌کند:

1. کاربر بعد از انتخاب سناریوی آماده خورشیدی مستقیماً وارد صفحه شرایط محیطی می‌شود.
2. در صفحه شرایط محیطی، شهر/منطقه از طریق بانک شهر، ورود دستی یا GPS ثبت می‌شود.
3. بعد از تأیید شرایط محیطی، مسیر دیگر به صفحه انتخاب مسیر پروژه برنمی‌گردد؛ مستقیم وارد لیست تجهیزات زیرشاخه خورشیدی می‌شود.
4. تجهیزات پیشنهادی سناریوی انتخاب‌شده به‌صورت خودکار در لیست تجهیزات فعال می‌شوند.
5. کاربر می‌تواند تجهیزات را حذف، اضافه و تعداد/ساعت مصرف/جزئیات تخصصی را اصلاح کند.
6. بعد از تأیید لیست تجهیزات، منطق کلی اپ ادامه پیدا می‌کند و کاربر وارد پیکربندی سیستم می‌شود.

این تغییر فقط در سناریوهای آماده خورشیدی فعال است و مسیر پروژه‌های دستی یا برق اضطراری را شلوغ نمی‌کند.
