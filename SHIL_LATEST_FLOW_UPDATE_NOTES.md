# SHIL Latest Flow Update Notes

این نسخه برای تست مرحله جدید آماده شده و شامل منطق موردنظر برای تغییر ترتیب Workflow است:

1. انتخاب مسیر پروژه به ابتدای مسیر منتقل شده است.
2. ترتیب جدید مسیر:
   - انتخاب مسیر پروژه
   - اطلاعات پروژه
   - شرایط محیطی
   - روش محاسبات
   - ورودی محاسبات
   - تنظیمات سیستم
   - خروجی نهایی
3. مسیر انتخاب‌شده پروژه در صفحه اطلاعات پروژه نمایش داده می‌شود.
4. مسیر انتخاب‌شده در بلوک نتایج اطلاعات پروژه نیز ثبت و نمایش داده می‌شود.
5. در صفحه ورودی محاسبات، داده‌های شرایط محیطی به‌صورت مستقل نمایش داده نمی‌شوند و داخل بلوک زمینه محاسبات ادغام می‌شوند.
6. بلوک زمینه محاسبات شامل روش انتخاب‌شده از شش مسیر محاسبات و پارامترهای مؤثر شرایط محیطی است.
7. ساختار ورودی محاسبات برای شش روش باید یکدست، بدون دیتای تکراری و دارای خروجی اختصاصی هر روش باشد.

مسیرهای محاسبات:
- لیست تجهیزات
- پروفایل مصرف
- انرژی روزانه
- توان پنل خورشیدی
- توان کل
- جریان کل
