# گزارش پیاده‌سازی سناریوهای آماده SHIL

## تغییرات انجام‌شده

- اضافه شدن صفحه مستقل «سناریوهای آماده SHIL SOLAR»
- اضافه شدن لینک «سناریوهای آماده» در نوار بالای داشبورد
- اضافه شدن دکمه «سناریوهای آماده» در Hero داشبورد
- اضافه شدن کارت دسترسی سریع «سناریوهای آماده»
- اضافه شدن دکمه دسترسی به سناریوها داخل Workspace
- اتصال هر سناریو به فرم پروژه و شروع طراحی با یک کلیک
- ثبت پکیج‌های آماده بر اساس تجهیزات برند SHIL
- استفاده از پنل‌های SHIL-550W، SHIL-620W و SHIL-700W
- استفاده از باتری‌ها و اینورترهای موجود در بانک تجهیزات SHIL

## سناریوهای اضافه‌شده

1. پکیج آپارتمان ۸۰ متری - آفگرید سبک
2. پکیج آپارتمان ۱۲۰ متری - آفگرید استاندارد
3. پکیج ویلای ۱۰۰ متری - آفگرید
4. پکیج ویلای ۱۸۰ متری - آفگرید پرمصرف
5. پکیج باغ و روشنایی محوطه
6. پکیج باغ با پمپ آب سبک
7. پکیج دفتر اداری ۸۰ متری - هیبرید
8. پکیج دفتر اداری ۱۵۰ متری - هیبرید
9. پکیج دفتر اداری ۲۵۰ متری - هیبرید حرفه‌ای
10. پکیج ویلای ۱۵۰ متری - هیبرید

## منطق عملکرد

با انتخاب هر پکیج:

- نوع سیستم تنظیم می‌شود؛ آفگرید یا هیبرید
- روش محاسبه مناسب انتخاب می‌شود
- مصرف روزانه یا لیست بارها پر می‌شود
- پنل، باتری و اینورتر SHIL انتخاب می‌شود
- ولتاژ سیستم، راندمان، DoD، تابش و سایر ورودی‌های اصلی تنظیم می‌شود
- کاربر به Workspace منتقل می‌شود و می‌تواند قبل از اجرای محاسبه، مقادیر را ویرایش کند

## تست انجام‌شده

دستور زیر اجرا شد و پاس شد:

```bash
npm run test:engineering:all
```

خروجی:

```text
Engineering smoke tests passed
Real engineering calculation tests passed
Engineering validation tests passed
All engineering test suites passed
```

## نکته اجرا

بعد از جایگزینی فایل‌ها در سیستم مقصد، برای مشاهده تغییرات جدید:

```bash
npm install
npm run dev
```

اگر نسخه قبلی در مرورگر دیده شد، کش PWA را پاک کنید یا Ctrl+Shift+R بزنید.
