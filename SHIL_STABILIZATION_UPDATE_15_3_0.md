# SHIL Stabilization Update 15.3.0

این نسخه برای نزدیک کردن اپ به بهره‌برداری واقعی ساخته شد.

## تغییرات کلیدی
- اتصال واقعی‌تر بانک‌های SHIL به موتور محاسبات.
- تغییر ترتیب Ruleها: انتخاب اینورتر پیش از پنل برای سازگاری MPPT و توان.
- انتخاب پنل بر اساس توان هدف، جریان MPPT، Voc سرد و محدوده اینورتر.
- انتخاب باتری بر اساس انرژی روزانه، ساعت پشتیبان، ولتاژ اینورتر و بانک SHIL.
- حفاظت PV، باتری و AC از بانک Protection انتخاب می‌شود و دیگر Placeholder خالی نیست.
- کابل‌ها از بانک کابل انتخاب و با جزئیات به Summary منتقل می‌شوند.
- خروجی تفکیکی هر اینورتر، MPPT، حفاظت، کابل و فضای نصب پایدارتر شد.
- بهبود Density، خوانایی، Footer، هدر، Summary و Result.

## تست پیشنهادی
```bash
npm install
npm run engine:smoke
npm run ops:check
npm run dev -- --force
```

قبل از تست مرورگر، Clear site data انجام شود.
