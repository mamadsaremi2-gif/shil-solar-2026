# نسخه V14 - Workflow UX + گزارش A4 اجرایی

## تغییرات اصلی

- حذف انتخاب پیش‌فرض مسیر و روش محاسبات برای پروژه‌های جدید.
- همه مراحل مسیر طراحی قابل مشاهده و قابل ورود هستند.
- دکمه تایید هر مرحله فقط وقتی فعال می‌شود که:
  - مرحله قبل تایید شده باشد.
  - فیلدهای اجباری همان صفحه معتبر باشند.
- اضافه شدن وضعیت مرحله‌ها در سایدبار:
  - قابل بازدید
  - نیازمند تایید مرحله قبل
  - تکمیل شده
  - فعال
- اضافه شدن پیام خطای واضح برای فیلدهای خالی یا ایراددار.
- مسیر پروژه و روش محاسبات دیگر با کلیک مستقیم وارد مرحله بعد نمی‌شوند؛ کاربر باید انتخاب را تایید کند.
- بازطراحی ظاهری کارت‌ها، کارت‌های مسیر طراحی، هدر صفحات و لوگوی ثابت صفحات.
- اضافه شدن انیمیشن ورود مرحله‌ها و نور فعال روی مرحله انتخاب‌شده.
- اصلاح خروجی PDF/PNG با روش تبدیل کامل گزارش A4 به تصویر برای جلوگیری از بهم‌ریختگی فارسی.
- اضافه شدن گزارش A4 اجرایی شامل:
  - مشخصات مشتری و پروژه
  - نیازها و روش محاسبات
  - پنل، اینورتر، باتری، تولید خالص
  - MPPT و شرایط نصب
  - همخوانی ولتاژ باتری
  - تجهیزات نصب و اجرا
  - پیام‌های مهندسی و اصلاحات لازم

## تست

- تست‌های مهندسی داخلی پاس شدند.
- بررسی سینتکس JSX انجام شد.
- Build در این محیط به‌دلیل نبودن وابستگی‌های نصب‌شده Vite اجرا نشد؛ روی سیستم کاربر بعد از `npm install` قابل build است.
