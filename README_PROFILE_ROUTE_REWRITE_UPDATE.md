# SHIL Profile Consumption Route Rewrite

این آپدیت مسیر «پروفایل مصرف» را از مسیر «لیست تجهیزات» و مسیر اختصاصی «توان پنل خورشیدی» جدا می‌کند.

## تغییرات
- حذف انتخاب تجهیزات از مسیر پروفایل مصرف.
- اضافه شدن ورودی مستقل برای مصرف صبح، ظهر، عصر و شب.
- اضافه شدن توان پیک مصرف، ولتاژ AC و ضریب راه‌اندازی مسیر پروفایل.
- اتصال خروجی پروفایل به موتور محاسبات بار عمومی بدون نمایش بانک پنل، MPPT یا تقسیم پنل.
- پاک شدن هشدار اشتباه «هیچ تجهیزی انتخاب نشده» در مسیرهای ورودی مستقیم.
- ذخیره `shil:profileConsumptionInput` برای استفاده در پیکربندی، چکیده و خروجی نهایی.

## محدودیت
این آپدیت ساختار مسیرهای دیگر را تغییر نمی‌دهد.
