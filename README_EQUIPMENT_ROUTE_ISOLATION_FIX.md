# SHIL Equipment Route Isolation Fix

این آپدیت فقط روی مسیر «لیست تجهیزات» و نشتی مسیر اختصاصی «توان پنل خورشیدی» تمرکز دارد.

## اصلاحات اصلی

- حذف نمایش بانک پنل، بانک اینورتر خورشیدی و بانک باتری اختصاصی مسیر توان پنل از مسیر لیست تجهیزات.
- حذف خروجی‌های MPPT، تقسیم پنل، آرایش پنل و تولید PV از مسیر لیست تجهیزات.
- خروجی پیکربندی مسیر لیست تجهیزات اکنون فقط بارمحور است: توان مصرفی، انرژی روزانه، جریان AC، جریان راه‌اندازی، پیک استارت، اینورتر پیشنهادی و باتری مرجع.
- چکیده اطلاعات برای مسیر لیست تجهیزات دیگر داده‌های PV را نمایش نمی‌دهد.
- خروجی نهایی اجرای محاسبات برای مسیر لیست تجهیزات دیگر جدول پنل/MPPT/PV تولید نمی‌کند.
- هنگام تأیید هر روش غیر از `solar_panel_power`، دیتای قدیمی `shil:solarPanelPowerInput` و `shil:solarPanelPowerPreview` پاک می‌شود تا localStorage باعث نشت UI نشود.

## محدوده اعمال

این اصلاح مخصوص مسیر `equipment` است و مسیر اختصاصی `solar_panel_power` همچنان بانک پنل، MPPT، تقسیم پنل و خروجی PV خودش را حفظ می‌کند.
