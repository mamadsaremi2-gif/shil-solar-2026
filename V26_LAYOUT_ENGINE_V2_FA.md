# SHIL V26 - Layout Engine V2

این نسخه روی پایه واقعی V24 ساخته شد و هدف آن اصلاح ریشه‌ای مشکل جابه‌جایی هدر و فوتر در موبایل است.

## تغییرات اصلی
- اضافه شدن `src/styles/v26-layout-engine-v2.css`
- ایمپورت شدن فایل V26 در انتهای `src/main.jsx`
- حذف وابستگی هدر/فوتر به `left: 50%` و `transform: translateX(-50%)`
- قفل شدن هدر و فوتر داخل عرض واقعی viewport
- حذف هدرها و فوترهای Legacy داخل صفحات داخلی
- حفظ داشبورد به عنوان صفحه مستقل
- اسکرول عمودی برای همه صفحات داخلی
- اسکرول افقی فقط در Stepbar، جدول‌ها، کارت‌های ردیفی و چارت‌ها
- مخفی شدن محتوا زیر هدر و فوتر ثابت با padding محاسبه‌شده

## نکته نصب
بعد از نصب و اجرا روی موبایل، اگر نسخه PWA کش شده باشد، Safari cache یا deployment cache را پاک کنید.
