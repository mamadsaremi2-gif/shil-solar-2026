# SHIL Solar Engineering V17 Final UI

این نسخه شامل بازطراحی نهایی ظاهر اپ بر اساس نمونه‌های تاییدشده است.

## موارد اعمال‌شده
- جایگزینی لوگوی SHIL IRAN در همه صفحات
- داشبورد سینمایی جدید با تصویر خانه خورشیدی و محصولات SHIL
- مسیر طراحی ۸ مرحله‌ای با ترتیب نهایی: اطلاعات پروژه، شرایط محیطی، انتخاب مسیر، روش محاسبات، ورودی محاسبات، تنظیمات سیستم، چکیده اطلاعات، اجرای محاسبات
- تم Dark Neon/Glassmorphism برای صفحات اصلی، فرم‌ها، کارت‌ها، جدول‌ها و دکمه‌ها
- بهبود صفحه سناریوهای آماده و لیست سناریوها
- بهبود صفحه ارتباط با ما با کارت محصولات و لینک SHIL.IR
- بهبود حالت چاپ و گزارش A4 با کنتراست مناسب برای چاپ سیاه‌وسفید
- واکنش‌گرایی موبایل و تبلت برای چیدمان‌های اصلی
