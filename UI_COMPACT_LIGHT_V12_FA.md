# SHIL SOLAR V12 - Compact Light UI Update

این نسخه قانون ظاهری سراسری اپ را اصلاح می‌کند:

- کارت‌ها و بلوک‌ها فشرده‌تر شدند.
- فاصله‌های خالی صفحات کاهش یافت.
- بلوک‌ها نزدیک‌تر شدند ولی با سایه و border از هم تفکیک می‌شوند.
- تم کلی از حالت بسیار تاریک به ظاهر روشن‌تر، مدرن‌تر و نزدیک‌تر به اپ‌های نصبی واقعی تغییر کرد.
- فرم‌ها، دکمه‌ها، جدول‌ها و کارت‌ها برای موبایل جمع‌وجورتر شدند.
- تغییرات فقط لایه UI/CSS است و موتور محاسبات، گزارش‌ها و داده‌های مهندسی تغییر نکرده‌اند.
