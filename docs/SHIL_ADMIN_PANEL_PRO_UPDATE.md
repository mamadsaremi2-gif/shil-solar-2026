# SHIL Admin Panel Pro Update

این آپدیت پنل مدیریت SHIL را برای نسخه لوکال/PWA تکمیل می‌کند.

## تغییرات اصلی

- اضافه شدن تب «کاربران» برای مشاهده کاربران دارای داده، تعداد پروژه‌ها، نظرات و پرسش‌ها.
- اضافه شدن تب «سیوها و پروژه‌ها» برای مشاهده، خروجی گرفتن، بستن/بازکردن و حذف رکوردهای ذخیره‌شده.
- اضافه شدن تب «نظرات و پاسخ‌ها» برای پاسخ مستقیم ادمین به نظرات کاربران و پرسش‌های دستیار.
- اضافه شدن مدیریت «یوزر و پسورد ورود ادمین» در تب امنیت.
- حفظ لایه دوم امنیت ادمین با PIN داخلی.
- اضافه شدن خروجی JSON برای کاربران، پروژه‌ها، نظرات و پرسش‌ها.

## مسیرهای مهم

- صفحه ادمین: `/admin`
- فایل اصلی پنل: `src/pages/AdminDashboard.jsx`
- منطق ورود ادمین: `src/auth/session.js`
- استایل‌های تکمیلی: `src/styles/shil-ui.css`

## ورود پیش‌فرض ادمین

در صورت ریست شدن تنظیمات:

- `admin` / `shil-admin`
- `admin@shil.app` / `shil-admin`
- `shil.admin` / `shil-admin`

رمز لایه دوم ادمین همچنان از بخش Admin Security کنترل می‌شود و مقدار پیش‌فرض لوکال آن `1366` است.
