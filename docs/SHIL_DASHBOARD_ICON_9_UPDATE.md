# افزودن آیکون ۹ داشبورد SHIL

## تغییر انجام‌شده

یک آیتم جدید با عنوان دقیق زیر به صفحه داشبورد اضافه شد:

```text
توسعه صفحه پروژه جدید
```

## مسیر مقصد

```text
/new-project/future
```

## فایل اصلاح‌شده

```text
src/data/shilFlowConfig.jsx
```

## شرایط رعایت‌شده

- عنوان دقیق طبق درخواست حفظ شد.
- آیتم به عنوان آیکون نهم داشبورد اضافه شد.
- مسیر به صفحه توسعه/آینده پروژه جدید متصل شد.
- استایل و ساختار فعلی داشبورد تغییر نکرد.
- از آیکون و Asset موجود بخش پروژه جدید استفاده شد تا وابستگی یا Asset جدید اضافه نشود.
