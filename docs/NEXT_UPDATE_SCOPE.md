# محدوده آپدیت بعدی

در آپدیت بعدی تمرکز روی طراحی ظاهری موبایل خواهد بود. زیرساخت آماده‌شده در v2 باید حفظ شود.

## مجاز در آپدیت بعدی

- طراحی Header/Footer ثابت
- پیاده‌سازی صفحات موبایل
- اتصال فرم‌ها به Workflow Store
- ساخت کارت‌ها، فیلدها، آیکون‌ها و مسیر طراحی
- بهینه‌سازی رفتار Scroll

## غیرمجاز بدون بررسی

- نوشتن محاسبات داخل UI
- تغییر قرارداد خروجی Engine
- حذف Pipeline
- حذف Schema Validation
- وابسته‌کردن Data Layer به Component
