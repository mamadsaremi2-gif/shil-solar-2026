# SHIL Scale Block Auto Visibility Update

این آپدیت منطق نمایش بلوک‌های مقیاس پروژه نیروگاهی را اصلاح می‌کند.

## رفتار جدید

- اگر توان طراحی مسیر محاسباتی کمتر یا مساوی 30kW باشد:
  - بلوک‌های نیروگاهی بسته می‌مانند.
  - فقط یک وضعیت فشرده نمایش داده می‌شود.
  - فیلدهای MV، ترانس، Grid Study، Tracker/GIS، SCADA و پارامترهای نیروگاهی صفحه را شلوغ نمی‌کنند.

- اگر توان طراحی از 30kW بالاتر برود یا طراح مقیاس صنعتی/نیروگاهی را دستی انتخاب کند:
  - بلوک کامل مقیاس پروژه نیروگاهی فعال می‌شود.
  - تنظیمات چنداینورتری، بلوک نیروگاهی، MV، ترانس، زمین، تولید سالانه، حفاظت، Grid Study و SCADA نمایش داده می‌شود.
