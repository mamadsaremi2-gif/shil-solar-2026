# گزارش پالایش جامع رابط کاربری

## تغییرات انجام‌شده

- حذف تمام تصاویر، گرادیان‌ها و لایه‌های بک‌گراند از ریشه برنامه و صفحات.
- غیرفعال‌سازی pseudo-elementهای تزئینی صفحات (`::before` و `::after`).
- حذف ظاهر کارت از wrapperهای صرفاً تزئینی: پس‌زمینه، سایه، blur و border اضافی.
- حذف خودکار کارت‌های واقعاً خالی پس از هر رندر و تغییر مسیر.
- کاهش فاصله‌های اضافی باقی‌مانده از کارت‌های حذف‌شده.
- حفظ کامل ورودی‌ها، فرم‌ها، دکمه‌ها، جدول‌ها، هشدارها، دیالوگ‌ها، نقشه‌ها و محل‌های نمایش پاسخ و نتیجه.

## روش اجرا

پاک‌سازی به‌صورت دو لایه انجام شده است:

1. `shil-comprehensive-ui-cleanup.css` برای خنثی‌کردن قطعی تمام استایل‌های قدیمی و تودرتو.
2. `shil-comprehensive-ui-cleanup.js` برای حذف کارت‌های خالی که به‌صورت پویا توسط React ساخته می‌شوند.

این دو فایل در انتهای `main.jsx` وارد شده‌اند تا بر همه themeها و overrideهای قبلی اولویت داشته باشند.

## استثناهای محافظت‌شده

موارد زیر حذف یا خنثی نشده‌اند:

- input / textarea / select / button
- جدول‌ها و خروجی‌های محاسباتی
- بخش‌های result / answer / output
- هشدارها و پیام‌های وضعیت
- دیالوگ‌ها و عناصر دسترس‌پذیری
- کارت‌هایی با `data-keep-card="true"` یا کلاس `shil-keep-card`
