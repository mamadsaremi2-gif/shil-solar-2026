# مسیر جایگزینی عکس‌های PNG صفحه انتخاب مسیر پروژه

برای تغییر عکس کارت‌های صفحه «انتخاب مسیر پروژه»، فایل‌های PNG را دقیقاً در مسیر زیر قرار دهید:

```text
public/assets/shil/execution/
```

نام فایل‌ها باید ثابت بماند:

```text
solar-execution.png
emergency-inverter-battery.png
utility-execution.png
```

نکته‌ها:

- کارت «اجرای پروژه با پنل خورشیدی» از `solar-execution.png` خوانده می‌شود.
- کارت «اجرای پروژه با برق اضطراری» از `emergency-inverter-battery.png` خوانده می‌شود.
- مسیر «نیروگاهی» به صورت فیلد بازشونده نمایش داده می‌شود و در صورت نیاز می‌تواند از `utility-execution.png` استفاده کند.
- فرمت پیشنهادی PNG با پس‌زمینه شفاف و نسبت تقریبی 4:3 یا 1.54:1 است.
- بعد از جایگزینی فایل‌ها، فقط Build یا Refresh انجام دهید؛ نیازی به تغییر کد نیست.
