# نسخه نهایی محاسبات و کتابخانه تجهیزات

این نسخه شامل موارد زیر است:

1. پشتیبانی از ورود اعداد فارسی/عربی در فیلدهای عددی.
2. نمایش واضح ستون «نام تجهیز» با عرض بیشتر و برچسب کنار همه سلول‌های بار.
3. افزودن کتابخانه تجهیزات خانگی پرتکرار با توان مصرفی، PF، ضریب همزمانی، نوع بار و ضریب راه‌اندازی.
4. افزودن بلوک‌های محاسباتی بار:
   - مجموع توان نصب‌شده تجهیزات
   - ضریب همزمانی میانگین
   - نیاز واقعی اجرا
   - انرژی روزانه واقعی
5. افزودن محاسبه برق اضطراری در حالت «سانورتر و باطری / بدون پنل خورشیدی»:
   - ورودی ساعت برق اضطراری موردنیاز مشتری
   - خروجی برق اضطراری واقعی قابل تأمین بر اساس تجهیزات و بانک باتری
   - نمایش آرایش باتری، تعداد کل باتری و انرژی قابل استفاده
6. اصلاحات مهندسی قبلی برای PV، باتری، MPPT، کابل و خروجی گزارش.

## نحوه اجرا بعد از جایگزینی

```bash
npm install
npm run build
npm run preview
```

## نکته مهم Supabase

فایل `.env.local` داخل ZIP قرار داده نشده است. بعد از جایگزینی، فایل `.env.local` قبلی خودت را نگه دار یا دوباره با مقادیر Supabase بساز.
