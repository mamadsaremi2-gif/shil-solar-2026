# نسخه V11 - فاز طراحی اجرایی واقعی

## تغییرات اصلی
- اینورتر قبل از PV محاسبه می‌شود و PV بر اساس محدودیت MPPT داخلی/خارجی اینورتر کنترل می‌شود.
- خروجی MPPT شامل تعداد MPPT، آرایش هر MPPT، Voc سرد، Vmp گرم، جریان کاری و Isc طراحی است.
- کنترل DC/AC Ratio اضافه شد.
- کابل DC/باتری/AC بر اساس دو معیار افت ولتاژ و جریان‌دهی حرارتی انتخاب می‌شود.
- ضرایب نصب کابل شامل دمای محیط، روش نصب، جنس کابل و گروه‌بندی مدارها اضافه شد.
- BOM حفاظتی شامل DC Isolator، SPD، String Fuse، Combiner Box، کابل PV، ارت، فیوز باتری و بریکر AC اضافه شد.
- Decision Engine اضافه شد تا طراحی را از نظر قابل اجرا بودن، اقتصادی بودن، بیش‌طراحی/کم‌طراحی و اصلاحات لازم ارزیابی کند.
- Energy Balance روز/شب اضافه شد: پوشش مستقیم خورشیدی، پوشش شب با باتری، مازاد PV، انرژی تامین‌نشده و self-consumption.
