# گزارش تغییرات سفارشی نسخه جدید SHIL Solar Design Suite

## تغییرات اصلی اعمال‌شده

- بازطراحی مرحله «نوع سیستم» به دو مسیر اصلی:
  - سیستم به همراه پنل خورشیدی
  - سیستم بدون پنل خورشیدی / برق اضطراری
- نمایش زیرمجموعه‌های آفگرید، هیبرید و آنگرید فقط پس از انتخاب مسیر پنل‌دار.
- تفکیک ورودی‌های محاسباتی:
  - مسیر پنل‌دار: روزهای خودمختاری
  - مسیر بدون پنل: زمان برق اضطراری مورد نیاز
- مقدار پیش‌فرض زمان برق اضطراری و روزهای خودمختاری روی صفر تنظیم شد.
- اضافه شدن گزینه «اینورتر دار / بدون اینورتر» برای بارهای لیستی.
- اصلاح محاسبه VA بارهای بدون اینورتر تا در سایزینگ اینورتر اثر مستقیم نگذارند.
- اضافه شدن دکمه بازگشت به صفحه قبل در Workspace و خروجی.
- اضافه شدن بخش ادامه پروژه‌های نیمه‌کاره در داشبورد.
- محدود شدن حذف پروژه و انتقال پروژه به سناریوهای آماده به مدیر اپ.
- ذخیره خودکار نسخه پروژه پس از اجرای موفق محاسبات.
- اضافه شدن توضیح خلاصه و قابل فهم برای علت تعداد، سری و موازی باتری‌ها.
- اضافه شدن توضیح فنی آرایش پنل شامل توان، ولتاژ، تعداد سری/موازی و علت انتخاب.
- اصلاح صفحه ارتباط با ما:
  - اطلاعات تماس قابل کپی
  - بلوک چهار QR آماده
  - بلوک ایتا، بله و روبیکا بدون QR
  - لینک برنامه قابل کپی و انتقال
- تنظیم فونت کلی اپ روی خانواده Vazir / Vazirmatn برای پایداری بهتر فارسی.

## نکته بیلد

در این محیط، نصب وابستگی‌های npm به دلیل timeout محیطی کامل نشد؛ بنابراین build نهایی Vite در همین محیط اجرا نشد. با این حال تست‌های محاسباتی Node.js پروژه پاس شدند. برای محیط نصب نهایی اجرا شود:

```bash
npm ci
npm run test:engineering:all
npm run build
```
