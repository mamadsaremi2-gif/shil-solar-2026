# SHIL Admin System 100% Update

این پکیج پنل ادمین را به نسخه کامل Frontend/Local Admin می‌رساند.

## تغییرات اصلی

- لایه ورود امن ادمین با PIN داخلی
- مدیریت کامل کارت‌های انتخاب مسیر پروژه
- امکان اضافه/حذف/فعال‌سازی کارت مسیر پروژه
- جایگزینی تصویر PNG برای کارت‌ها بدون تغییر کد
- مدیریت بانک پنل، اینورتر خورشیدی، باتری، برق اضطراری و حفاظت‌ها
- مدیریت پیش‌فرض‌های مهندسی، پنل ۶۲۰ وات و زمان برق اضطراری مورد نیاز
- Import / Export کامل JSON
- نسخه پشتیبان دستی و خودکار
- بازیابی Snapshot
- Health Check داخلی برای کنترل معتبر بودن تنظیمات انتشار
- لاگ تغییرات ادمین
- پشتیبانی مسیر توسعه با پیام در حال توسعه
