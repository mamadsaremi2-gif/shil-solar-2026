# یادداشت اصلاحات ظاهری موبایل V4

این Patch بدون تغییر منطق محاسبات و ساختار داده، فقط لایه ظاهری/رفتاری موبایل را اصلاح می‌کند.

## اصلاحات انجام‌شده

- یکپارچه‌سازی Header ثابت در همه صفحات موبایل
- یکپارچه‌سازی Footer ثابت و چسباندن کامل آن به پایین صفحه
- مخفی شدن محتوای اسکرول‌شونده زیر Header / Footer به‌جای افتادن روی نوارها
- حذف اسکرول افقی سراسری صفحه
- حفظ اسکرول افقی فقط برای مسیر طراحی
- فشرده‌سازی فاصله کارت‌ها، فرم‌ها و بلوک‌های دیتاسنگین
- بزرگ‌تر و متناسب‌تر شدن کارت‌های انتخاب مسیر PV و برق اضطراری
- اصلاح نمایش نقشه ایران با `object-fit: contain` و نسبت تصویر کنترل‌شده
- کاهش blur/glowهای سنگین برای بهبود Performance موبایل
- فشرده‌سازی صفحه ارتباط با ما و جلوگیری از فضای پرت
- بهبود لمس دکمه‌ها با فیدبک سبک شبیه کلید مکانیکی
- آماده‌سازی Safe Area برای گوشی‌های دارای notch و navigation bar
- سبک‌سازی تصاویر اصلی موبایل بدون تغییر مسیر فایل‌ها

## فایل‌های تغییرکرده

- `src/main.jsx`
- `src/styles/mobile-refinement-patch-v4.css`

## نکته

این Patch عمداً به انتهای زنجیره CSS اضافه شده تا بدون تخریب فایل‌های قبلی، نقش override نهایی را داشته باشد.

## بهینه‌سازی تصاویر

تصاویر سنگین اصلی کارت مسیر، بک‌گراند داشبورد، لوگو و نقشه محیطی کم‌حجم شدند تا بارگذاری موبایل سریع‌تر شود. مسیر فایل‌ها تغییر نکرده و نیازی به تغییر کد نیست.
