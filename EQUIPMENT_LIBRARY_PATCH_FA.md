# اصلاح کتابخانه تجهیزات خانگی

این نسخه شامل اضافه شدن ۵۱ مصرف‌کننده خانگی از فایل اکسل آپلود شده به کتابخانه تجهیزات است.

## تغییرات
- اضافه شدن تجهیزات خانگی پرتکرار شامل کولر گازی، کولر آبی، یخچال، لوازم آشپزخانه، روشنایی، مودم، پمپ، کف‌کش و سایر موارد.
- هر تجهیز دارای توان نامی، ضریب توان، نوع بار، ضریب همزمانی، ساعت کار پیشنهادی و ضریب راه‌اندازی اولیه است.
- جستجو اکنون اعداد فارسی و انگلیسی را بهتر پیدا می‌کند؛ مثلا جستجوی 12000 و ۱۲۰۰۰ هر دو به نتیجه می‌رسند.
- عرض فیلد نام تجهیز در فرم بارها اصلاح شد تا نام تجهیز واضح دیده شود.

بعد از جایگزینی فایل‌ها، اجرا کنید:

```bash
npm install
npm run build
npm run preview
```
