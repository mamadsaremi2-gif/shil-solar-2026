# گزارش افزودن تست‌های مهندسی واقعی

این نسخه علاوه بر Smoke Test قبلی، یک مجموعه تست مهندسی واقعی‌تر دارد که خروجی محاسبات را با فرمول‌ها، حدود قابل قبول و سناریوهای کاربردی کنترل می‌کند.

## فایل‌های اضافه‌شده

- `scripts/engineering-real-tests.mjs`
- `scripts/run-engineering-tests.mjs`

## اسکریپت‌های جدید

```bash
npm run test:engineering:real
npm run test:engineering:all
```

## پوشش تست‌ها

### REAL-01 — سیستم Off-grid ویلایی 5kWh/day
کنترل می‌کند:
- انرژی روزانه و توان تقاضا
- ظرفیت Ah باتری بر اساس فرمول مستقل
- تعداد سری/موازی باتری
- تعداد پنل، توان نصب‌شده، PR و رشته‌بندی MPPT
- سایز اینورتر، کنترلر و افت ولتاژ کابل باتری

### REAL-02 — بار موتوری و Surge
کنترل می‌کند:
- محاسبه مصرف روزانه بارهای لیستی
- لحاظ شدن جریان/توان راه‌اندازی موتور
- پوشش توان لحظه‌ای و VA توسط اینورتر
- ایمنی C-rate باتری
- هشدار ضریب توان پایین

### REAL-03 — دمای پایین و محدوده MPPT
کنترل می‌کند:
- افزایش Voc پنل در سرما
- جلوگیری از عبور Voc رشته از حداکثر کنترلر
- تولید هشدار برای خروج از پنجره MPPT
- تغییر وضعیت طراحی به warning

### REAL-04 — سیستم Backup-only
کنترل می‌کند:
- عدم تولید آرایه پنل و کنترلر برای حالت Backup
- پوشش زمان بکاپ درخواستی
- آرایش صحیح بانک باتری 24V با باتری 12V
- صفر بودن کابل PV DC

### REAL-05 — سیستم Grid-tie با هدف جبران انرژی
کنترل می‌کند:
- نبود باتری در طراحی Grid-tie
- وجود آرایه PV
- رعایت درصد offset انرژی
- پوشش تولید روزانه نسبت به هدف طراحی

## دستور پیشنهادی برای کنترل نهایی

```bash
npm ci
npm run test:engineering:all
npm run build
```
