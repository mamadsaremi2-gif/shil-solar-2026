# SHIL Emergency Ready Scenario Flow Update

این آپدیت مسیر سناریوهای آماده برق اضطراری را با مسیر سناریوهای آماده خورشیدی هم‌ساختار می‌کند، اما هسته محاسباتی و خروجی‌ها را در شاخه برق اضطراری نگه می‌دارد.

## مسیر جدید

```txt
سناریوی آماده برق اضطراری
→ شرایط محیطی / GPS / شهر / منطقه
→ لیست تجهیزات زیرشاخه برق اضطراری
→ حذف، اضافه و اصلاح تعداد و ساعت مصرف تجهیزات
→ تایید تجهیزات
→ پیکربندی سیستم برق اضطراری
→ ادامه مسیر تا چکیده و خروجی نهایی
```

## تغییرات فنی

- صفحه `Environment.jsx` اکنون هم برای `solar` و هم برای `emergency`، در حالت سناریوی آماده، پس از تایید شرایط محیطی مستقیماً به لیست تجهیزات همان شاخه هدایت می‌کند.
- صفحه `CalculationInputs.jsx` اکنون سناریوی آماده برق اضطراری را هم مثل سناریوی آماده خورشیدی seed می‌کند.
- تجهیزات پیشنهادی سناریو در حالت برق اضطراری از بانک تجهیزات مصرف‌کننده استخراج و از قبل انتخاب می‌شوند.
- حذف، اضافه، اصلاح تعداد، ساعت مصرف، ضریب همزمانی، PF و سافت‌استارتر در همان صفحه حفظ شده است.
- خروجی نهایی مسیر با `domain=emergency` وارد موتور محاسبات برق اضطراری می‌شود، نه موتور خورشیدی.

## بدون تغییرات غیرمهندسی

هیچ بخش قیمت، خرید، فروش یا مارکت‌پلیس اضافه نشده است.
