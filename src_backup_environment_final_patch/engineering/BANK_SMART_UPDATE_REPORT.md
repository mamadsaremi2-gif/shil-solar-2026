# SHIL Smart Equipment Bank Update

این نسخه روی بانک پنل، اینورتر و باتری و نمایش آن‌ها در اپ اعمال شده است.

## 1. بانک پنل خورشیدی

- بانک پنل از حالت کلاس‌های بازه‌ای خارج شد.
- مقادیر محاسباتی هر پنل اکنون مقدار دقیق دیتاشیت هستند، نه بازه از/تا.
- پنل پیش‌فرض هوشمند روی مدل 620W تنظیم شد.
- پنل‌های توان پایین فقط به‌عنوان حالت خاص نگه داشته شدند.
- مسیر «توان پنل خورشیدی» در `CalculationInputs.jsx` اکنون از بانک مرکزی `src/engineering/bank` می‌خواند.
- صفحه تنظیمات سیستم، دیتاشیت خلاصه پنل را نمایش می‌دهد: Pmax, Vmp, Voc, Imp, Isc, Cell Type, Efficiency, Dimensions, Use Case.

## 2. بانک اینورتر خورشیدی

- انتخاب پیش‌فرض ثابت 1.6KW حذف شد.
- اینورتر براساس توان طراحی، ضریب اطمینان، نوع سیستم و قابلیت پارالل انتخاب می‌شود.
- در سیستم‌های آفگرید/هیبرید، اینورترهای On-Grid/Utility به‌عنوان انتخاب پیش‌فرض وارد نمی‌شوند.
- اگر توان طراحی از ظرفیت یک اینورتر عبور کند، تعداد اینورتر موازی در خروجی ثبت و هشدار داده می‌شود.
- دیتاشیت اینورتر در UI نمایش داده می‌شود و پارامترهای MPPT, PV Voc, PV Current, Max PV Power و Parallel Capability روی محاسبات اثر دارند.

## 3. بانک باتری

- باتری از روی ولتاژ DC اینورتر انتخاب می‌شود.
- برای اینورتر 48V، پیش‌فرض هوشمند روی باتری LiFePO4 51.2V 100Ah قرار می‌گیرد.
- اگر کاربر باتری 12V یا 24V انتخاب کند، موتور آرایش سری/موازی را محاسبه می‌کند.
- تعداد کل باتری براساس روز/ساعت خودکفایی، انرژی روزانه، DoD و راندمان محاسبه می‌شود.
- آرایش سری/موازی، ولتاژ پک، تعداد کل و انرژی کل در خروجی سیستم و UI ثبت می‌شود.

## 4. مسیر برق اضطراری

- صفحه تنظیمات برق اضطراری اکنون از همان بانک اینورتر خورشیدی و همان بانک باتری مرکزی استفاده می‌کند.
- انتخاب هوشمند اینورتر و باتری با منطق ولتاژ DC و سری/موازی باتری هماهنگ شد.
- نمایش دیتاشیت برای اینورتر و باتری در صفحه برق اضطراری اضافه شد.
