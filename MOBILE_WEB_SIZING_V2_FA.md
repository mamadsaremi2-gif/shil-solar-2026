# SHIL SOLAR Mobile/Web Responsive Sizing V2

این نسخه قوانین سایزینگ نسخه موبایل/وب را به صورت سراسری روی اپ اعمال می‌کند:

- موبایل تا 600px: یک ستون، متن 14 تا 16px، کارت‌های Stacked، فقط اسکرول عمودی
- تبلت 600 تا 900px: دو ستون، متن 16 تا 18px
- دسکتاپ بالای 900px: سه تا چهار ستون، متن 18 تا 20px
- کارت‌ها Auto-height هستند و ارتفاع ثابت ندارند
- فاصله داخلی کارت‌ها: 12px موبایل و 20px دسکتاپ
- فاصله کارت‌ها: 8px موبایل و 12px دسکتاپ
- فاصله سکشن‌ها: 24px موبایل و 32px دسکتاپ
- جدول‌ها در موبایل Stacked می‌شوند
- جزئیات پیشرفته در موبایل مخفی/Collapsible و در دسکتاپ باز هستند
- موبایل از Bottom Navigation و دسکتاپ از Sidebar Navigation پشتیبانی می‌کند
- اسکرول افقی سراسری بسته شده است

فایل اصلی تغییر یافته:

```txt
src/styles/mobile-standalone-v1.css
```
