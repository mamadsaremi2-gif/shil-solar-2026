# SHIL Full Offline PWA Update

این آپدیت روی نسخه قبلی اعمال شده و هدف آن اجرای ۱۰۰٪ آفلاین برای بخش‌های مهندسی اپ است.

## اضافه‌شده‌ها

- پیکربندی کامل `vite-plugin-pwa`
- کش کامل Routeها، JS، CSS، تصاویر، فونت‌ها و فایل‌های عمومی
- Route fallback برای Refresh در حالت آفلاین
- ذخیره پروژه‌ها با `localforage`
- ذخیره تصاویر آپلودی به صورت DataURL در Offline Storage
- صف عملیات آفلاین برای Sync آینده
- API داخلی `window.SHIL_OFFLINE`
- نشانگر آنلاین/آفلاین داخل اپ
- Manifest کامل برای نصب موبایل

## تست پیشنهادی

```bash
npm install
npm run build
npm run preview
```

بعد اپ را یک بار باز کنید، اینترنت را قطع کنید و Refresh بزنید. اگر صفحه‌ها و محاسبات بالا آمدند، حالت آفلاین عملیاتی است.

## نکته

داده‌هایی که به API خارجی وابسته باشند در حالت آفلاین فقط با آخرین کش یا داده محلی کار می‌کنند. موتورهای محاسباتی SHIL چون داخل JS هستند، آفلاین اجرا می‌شوند.
