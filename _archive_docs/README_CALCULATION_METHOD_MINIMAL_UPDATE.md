# SHIL - Calculation Method Page Update

تغییرات اعمال‌شده:

1. بعد از تأیید صفحه انتخاب مسیر پروژه، مسیر `/new-project/method` صفحه انتخاب روش محاسبات را نمایش می‌دهد.
2. صفحه روش محاسبات دارای ۵ کارت مینیمال است.
3. توضیحات ریز و متن‌های شلوغ از کارت‌ها حذف شد.
4. کاربر با انتخاب هر کارت مستقیم وارد صفحه ورودی همان روش می‌شود:
   `/new-project/input/:domain/:method`
5. لیست کارت‌ها از ریشه public قابل تغییر است:
   `public/calculation-method-cards.json`

برای تغییر عنوان یا badge کارت‌ها فقط فایل JSON بالا را ویرایش کنید.
