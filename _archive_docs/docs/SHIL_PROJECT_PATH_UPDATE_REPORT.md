# گزارش آپدیت صفحه انتخاب مسیر پروژه

## تغییرات انجام‌شده

- متن توضیحی «مسیرهای معمولی مرحله‌به‌مرحله...» کامل حذف شد.
- دو کارت اصلی در جای خود باقی ماندند:
  - اجرای پروژه با پنل خورشیدی
  - اجرای پروژه با برق اضطراری
- مسیر نیروگاهی از کارت اصلی به فیلد بازشونده منتقل شد تا فضای صفحه را نگیرد.
- کنتراست بک‌گراند، کارت‌ها و متن‌ها اصلاح شد تا متن‌ها خواناتر باشند.
- مسیر تصاویر کارت‌ها از SVG به PNG قابل‌تعویض تغییر کرد.
- راهنمای جایگزینی عکس‌ها در مسیر زیر اضافه شد:
  - `public/assets/shil/execution/README_ADMIN_REPLACE_IMAGES.md`

## مسیر و نام فایل‌های PNG قابل جایگزینی

```text
public/assets/shil/execution/solar-execution.png
public/assets/shil/execution/emergency-inverter-battery.png
public/assets/shil/execution/utility-execution.png
```

## نکته تست

بعد از جایگزینی PNGها، فقط Refresh یا Build کافی است و نیازی به تغییر کد نیست.
