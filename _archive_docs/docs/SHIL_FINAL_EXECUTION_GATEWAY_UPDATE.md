# SHIL Final Execution Gateway Update

این نسخه آخرین تغییرات مسیرهای بررسی‌شده را روی پکیج قبلی اعمال می‌کند.

## تغییرات اصلی

- اضافه‌شدن صفحه «نوع اجرای پروژه» بعد از تأیید یکی از روش‌های محاسبات بار.
- نمایش دو کارت تصویری:
  - اجرای پروژه با پنل خورشیدی
  - اجرای پروژه با برق اضطراری
- انتخاب کارت فقط پس از تأیید مرحله به صفحه بعد منتقل می‌شود.
- مسیر برق اضطراری در UI فقط با عنوان «برق اضطراری با اینورتر و باتری» نمایش داده می‌شود.
- نام UPS در رابط کاربری حذف شد؛ مدل محاسباتی داخلی همچنان بر پایه قوانین پشتیبان باتری/اینورتر کار می‌کند.
- عبارت «زمان بکاپ» به «زمان برق اضطراری مورد نظر» تغییر کرد.
- خروجی Load Engine بعد از روش‌های لیست تجهیزات، توان کل، جریان کل، انرژی مورد نیاز و پروفایل مصرف به این Gateway منتقل می‌شود.

## مسیر جدید

```txt
شرایط محیطی
↓
روش محاسبات
↓
ورودی بار / توان / جریان / انرژی / پروفایل
↓
نوع اجرای پروژه
↓
طراحی سیستم خورشیدی یا برق اضطراری
```
