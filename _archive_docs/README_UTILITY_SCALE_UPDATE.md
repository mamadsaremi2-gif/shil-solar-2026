# SHIL Utility Scale Update

این آپدیت منطق مقیاس پروژه را به موتور مهندسی SHIL اضافه می‌کند.

## محدوده جدید
- طراحی کوچک: تا 30kW
- تجاری/صنعتی سبک: 30kW تا 500kW
- صنعتی بزرگ: 500kW تا 2MW
- نیروگاهی: 2MW تا 10MW
- نیروگاهی بزرگ: 10MW تا 30MW

## رفتار مهندسی
اگر ظرفیت از اینورتر تکی 30kW بالاتر باشد، اپ خطا نمی‌دهد. طراحی به یکی از حالت‌های زیر تبدیل می‌شود:

- single_inverter
- multi_inverter
- block_based_power_plant

## نکته مهم
این آپدیت هیچ منطق قیمت، خرید، فروش، فروشگاه یا مارکت‌پلیس اضافه نمی‌کند. خروجی‌ها فقط برای تحلیل مهندسی، اعتبارسنجی، بلوک‌بندی، تعداد اینورتر، توان DC/AC، پنل، حفاظت و گزارش فنی هستند.
