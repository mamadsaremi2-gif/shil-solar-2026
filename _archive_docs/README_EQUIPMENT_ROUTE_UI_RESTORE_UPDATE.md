# Equipment Route UI Restore Update

این آپدیت مسیر «لیست تجهیزات» را به ساختار موردنظر کاربر بازگرداند:

- کنترل طراحی عمومی و انتخاب نوع اجرا باقی می‌ماند.
- پارامترهای اثرگذار شامل روزهای خودکفایی و ضریب اطمینان استاندارد نمایش داده می‌شود.
- بانک‌های عمومی تجهیزات شامل بانک اینورتر، بانک باتری و بانک پنل در مسیر لیست تجهیزات فعال هستند.
- بخش‌های اختصاصی مسیر توان پنل خورشیدی مانند MPPT اختصاصی، تقسیم پنل بین اینورترها و جدول PV Route فقط برای `solar_panel_power` باقی می‌مانند.
- نتیجه پیکربندی مسیر لیست تجهیزات همچنان بارمحور است و از `GeneralLoadResultTable` استفاده می‌کند.
