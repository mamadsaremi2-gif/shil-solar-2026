# SHIL Frontend UI 100% Update

این آپدیت فقط لایه Frontend/UI را نهایی می‌کند و موتورهای محاسباتی را تغییر نمی‌دهد.

## موارد اعمال‌شده
- یکپارچه‌سازی رفتار Header/Footer و فضای امن پایین صفحه.
- جلوگیری از overflow افقی در صفحات اصلی.
- پشتیبانی نوشتاری فارسی/انگلیسی و اعداد فارسی/انگلیسی در فیلدهای متنی.
- بزرگ‌تر شدن آیکون‌های داشبورد و پروژه جدید بدون خروج از کادر.
- بهبود کارت‌ها، دکمه‌ها، جداول و خروجی‌های نهایی با سبک Industrial/Banking UI.
- تثبیت رفتار دکمه‌های تأیید در انتهای اسکرول، نه شناور ناخواسته.
- آماده‌سازی جایگزینی عکس کارت‌های مسیر پروژه از طریق `public/project-path-cards.json` و فایل‌های PNG در `public/assets`.
- ساده‌سازی صفحه «توسعه» با متن «در حال توسعه» و دکمه برگشت.
