# SHIL System Settings Final UI Update

این نسخه صفحه «پیکربندی تجهیزات سیستم» را بازطراحی و تکمیل می‌کند:

- کارت‌های کنترل طراحی برای آفگرید، آنگرید و هیبرید با حالت انتخاب فعال.
- پارامترهای اثرگذار روز خودکفایی و ضریب افزایش استاندارد که مستقیماً به موتور انتخاب تجهیزات وصل هستند.
- بانک‌های اینورتر، باتری و پنل در بلوک‌های جدا و فشرده.
- ورود دستی تجهیزات با خروج از حالت اعمال هوشمند.
- ضریب/تعداد توسعه آینده برای هر بانک و اثر مستقیم روی محاسبات.
- توضیحات هر بانک به صورت باز/بسته برای جلوگیری از شلوغی صفحه.
- جدول نتیجه پیکربندی ساختاریافته با ستون بخش، مقدار و دلیل.
- بلوک «نمایش جزئیات جدول، کابل و حفاظت» چسبیده به جدول نتیجه.
- گزارش تفکیکی انتخاب کابل، حفاظت، فضا، سری/موازی و دلیل انتخاب هوشمند.
- دکمه تأیید نهایی به صفحه چکیده.
