# SHIL Engineering Logic 100% Update

این پکیج هسته محاسبات مهندسی را از حالت نمایشی/نیمه‌کامل به ساختار مستقل، قابل اعتبارسنجی و قابل استفاده در خروجی نهایی ارتقا می‌دهد.

## تغییرات اصلی

- موتور خورشیدی با خروجی مهندسی کامل‌تر:
  - انتخاب هوشمند ولتاژ سیستم 12/24/48V
  - پنل پیش‌فرض 620 وات و پشتیبانی از انتخاب دستی 700 وات
  - محاسبه آرایش سری/موازی پنل با کنترل MPPT، Voc سرد و Vmp گرم
  - محاسبه بانک باتری با DoD، راندمان، سری و موازی
  - اعمال ضرایب توسعه آینده روی پنل، اینورتر و باتری
  - محاسبه کابل و حفاظت DC/AC/باتری
  - خروجی Validation، Warnings، Errors، Confidence و Equipment Schedule

- موتور برق اضطراری مستقل:
  - بدون نمایش واژه UPS در UI
  - استفاده از عبارت «زمان برق اضطراری مورد نیاز»
  - محاسبه توان دائم، توان لحظه‌ای، اینورتر برق اضطراری، باتری و حفاظت‌ها
  - خروجی مستقل برای تجهیزات حفاظتی، بانک باتری، کابل‌ها و اعتبارسنجی
