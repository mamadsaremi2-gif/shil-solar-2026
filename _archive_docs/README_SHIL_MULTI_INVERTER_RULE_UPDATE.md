# SHIL Multi-Inverter Rule Update

این نسخه قانون مشترک چنداینورتری را برای تمام مسیرهای ورود از صفحه انتخاب روش محاسبات اعمال می‌کند:

- لیست تجهیزات
- پروفایل مصرف
- انرژی روزانه
- توان پنل خورشیدی
- توان کل
- جریان کل

## قانون اصلی
اگر توان/مدل اینورتر انتخاب یا دستی وارد شود، موتور محاسبات:

1. توان طراحی را بر اساس ورودی همان مسیر محاسبه می‌کند.
2. تعداد اینورتر لازم را از تقسیم توان طراحی بر توان هر اینورتر به دست می‌آورد.
3. پنل، باتری، حفاظت AC/DC، کابل و فضای نصب را برای هر اینورتر جداگانه محاسبه می‌کند.
4. خروجی نهایی را هم به صورت زیرسیستم‌های مستقل `INV-01`, `INV-02`, ... و هم به صورت جمع کل نمایش می‌دهد.

## اصلاحات مهم
- برای پروژه‌های چنداینورتری، توان PV فقط مساوی توان مصرفی خام نیست و با نسبت DC/AC و ظرفیت ورودی اینورتر کنترل می‌شود.
- باتری‌ها بین اینورترها به صورت توزیع عدد صحیح تقسیم می‌شوند؛ مثال: ۹ باتری بین ۵ اینورتر به صورت ۲/۲/۲/۲/۱.
- حفاظت و کابل هر اینورتر مستقل گزارش می‌شود.
- جدول خروجی نهایی و PDF، زیرسیستم‌های هر اینورتر را نمایش می‌دهد.
