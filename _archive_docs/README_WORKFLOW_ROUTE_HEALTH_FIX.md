# SHIL Workflow Route Health Fix

این آپدیت برای رفع برگشت ناگهانی مسیر «روش محاسبات > لیست تجهیزات» به صفحه انتخاب مسیر پروژه اعمال شد.

## اصلاحات اصلی

- Route Guard دیگر مسیر دستی `/new-project/input/<domain>/equipment` را با سناریوی آماده اشتباه نمی‌گیرد.
- مسیر لیست تجهیزات سناریوی آماده فقط وقتی Guard می‌شود که query واقعاً سناریویی باشد، مثل `from=scenario` یا `scenarioId`.
- صفحه انتخاب مسیر پروژه در حالت دستی دیگر `scenarioDomain` را ست نمی‌کند تا State سناریوی قبلی وارد پروژه جدید نشود.
- صفحه ورودی محاسبات ابتدا از `params.domain` و سپس از `calculationDomain` استفاده می‌کند تا مسیر دستی پایدار بماند.
- مسیر نیروگاهی همچنان فقط از کارت نیروگاهی یا هدایت هوشمند فعال می‌شود.

## نتیجه مورد انتظار

- روش «لیست تجهیزات» در پروژه جدید معمولی تا صفحه پیکربندی سیستم ادامه می‌دهد.
- سناریوی آماده خورشیدی و برق اضطراری همچنان بعد از شرایط محیطی وارد لیست تجهیزات سناریو می‌شوند.
- مسیرهای توان کل، جریان کل، انرژی روزانه و توان پنل خورشیدی بدون تغییر ساختاری باقی می‌مانند.
