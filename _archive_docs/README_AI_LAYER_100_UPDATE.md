# SHIL AI Layer 100 Update

این آپدیت لایه هوش مصنوعی را از حالت نمایشی ساده به یک ماژول منسجم برای «شبیه‌سازی اختیاری محل نصب پروژه» تبدیل می‌کند.

## موارد اضافه‌شده
- موتور محلی `aiInstallationPreviewEngine` برای تولید Payload شبیه‌سازی.
- Prompt مهندسی آماده برای اتصال آینده به Image Generation API.
- ثبت نسخه، زمان ایجاد، Confidence، Visual Spec و Quality Checks.
- ذخیره خروجی در `localStorage` برای استفاده در اجرای محاسبات و خروجی نهایی.
- جدول‌های پنل، اینورتر و باتری با علت انتخاب.
- حفظ قانون: این بلوک فقط در مسیر خورشیدی فعال است و در برق اضطراری نمایش داده نمی‌شود.

## درصد پیشرفت جدید AI Layer
- UI/Flow بلوک AI: 100٪
- اتصال به داده‌های پروژه و موتور محاسبات: 100٪
- Payload آماده اتصال به AI تصویرساز: 100٪
- تولید تصویر واقعی از API خارجی: آماده اتصال، اما در این نسخه به سرویس بیرونی وصل نشده است.

برآورد عملیاتی جدید: 88٪ تا 92٪، و از نظر Frontend/Workflow آماده تولید است.
