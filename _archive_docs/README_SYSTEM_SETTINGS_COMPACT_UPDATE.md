# SHIL System Settings Compact Update

این نسخه صفحه پیکربندی تجهیزات سیستم را مطابق قوانین جدید اصلاح می‌کند:

- کنترل طراحی به سه کارت آفگرید، آنگرید و هیبرید تبدیل شد.
- روز خودکفایی و ضریب افزایش استاندارد در بلوک جدا قرار گرفتند و مستقیم وارد موتور انتخاب تجهیزات می‌شوند.
- بانک اینورتر، باتری و پنل داخل سه بلوک مستقل قرار گرفتند.
- کاربر می‌تواند انتخاب هوشمند SHIL را فعال کند یا با ورود دستی تجهیزات و تعداد، از انتخاب خودکار خارج شود.
- توضیحات کنار بانک‌ها به فیلد باز/بسته‌شونده منتقل شد تا صفحه شلوغ نشود.
- نتیجه پیکربندی به جدول مرتب و قابل بررسی تبدیل شد.
- گزارش دلیل انتخاب کابل و سیستم حفاظتی داخل فیلد جزئیات قرار گرفت.
- تأیید نهایی همچنان فقط در صورت معتبر بودن منطق محاسباتی انجام می‌شود و سپس کاربر به صفحه چکیده می‌رود.
