# SHIL Equipment Input Update

اعمال شد:

- لیست ۲۵۰ تجهیز داخل یک فیلد باز/بسته‌شونده قرار گرفت.
- کاربر می‌تواند با سرچ کلمه‌ای یا اسکرول تجهیزات را ببیند.
- با کلیک روی هر تجهیز، بلافاصله انتخاب/اعمال می‌شود.
- بعد از انتخاب، کاربر می‌تواند لیست را ببندد و تجهیزات انتخابی را جداگانه ببیند.
- امکان اصلاح تعداد، ساعت مصرف، ضریب همزمانی، راندمان/PF و حذف تجهیز اضافه شد.
- دکمه «اعمال هوشمند» اطلاعات ساده را به فیلدهای تخصصی تبدیل می‌کند.
- بارهای موتوری به‌صورت پیش‌فرض با ضریب جریان راه‌اندازی ۲.۵ برابر جریان نامی محاسبه می‌شوند.
- اگر سافت‌استارتر فعال شود، ضریب راه‌اندازی همان تجهیز به ۱.۲ برابر جریان نامی تغییر می‌کند.
- این ضرایب در موتور اصلی بار و خروجی‌های توان، جریان راه‌اندازی و پیک استارت اثر مستقیم دارند.
- بعد از تأیید این صفحه، مسیر به صفحه پیکربندی تنظیمات می‌رود: `/new-project/system/:domain`.
