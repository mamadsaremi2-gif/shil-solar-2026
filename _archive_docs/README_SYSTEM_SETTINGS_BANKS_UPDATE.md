# SHIL System Settings Banks Update

این نسخه صفحه پیکربندی تنظیمات را با ۳ بانک انتخاب تجهیزات SHIL به‌روزرسانی می‌کند:

- بانک اینورتر خورشیدی از ۱.۶ تا ۳۰ کیلووات
- بانک باتری ۱۲/۲۴/۴۸ ولت با ظرفیت‌های مشخص‌شده
- بانک پنل خورشیدی ۴۰۰، ۴۵۰، ۵۵۰، ۶۲۰ و ۷۰۰ وات

منطق اضافه‌شده:
- پیشنهاد هوشمند اینورتر، باتری و پنل بر اساس توان و انرژی مصرفی
- هشدار ناسازگاری انتخاب دستی با توان مصرفی
- تعداد مجزا برای پنل، اینورتر و باتری جهت توسعه آینده
- محاسبه سری/موازی پنل و باتری با دلیل
- گزارش کابل، تجهیزات حفاظتی و فضای نصب
- رعایت سازگاری ولتاژ شناور باتری با ورودی ثابت اینورتر
