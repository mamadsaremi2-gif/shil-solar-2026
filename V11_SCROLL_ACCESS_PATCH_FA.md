# گزارش نسخه V11 Scroll Access Patch

این نسخه بر پایه نسخه پایدار V11 ساخته شده و فقط یک Patch محدود برای دسترسی بهتر به اسکرول اضافه شده است.

## تغییرات انجام‌شده

- اضافه شدن فایل `src/styles/v11-scroll-access-patch.css`
- اتصال Patch به ورودی اصلی CSS در `src/styles/modular-appearance.css`
- باز شدن اسکرول عمودی برای صفحات موبایل، داشبورد، ارتباط با ما، پروژه‌ها، سناریوها، آموزش، بازخورد و workspace
- باز شدن اسکرول عمودی داخلی برای کارت اصلی مراحل طراحی بین Header، Stepbar و Footer ثابت
- باز شدن اسکرول افقی فقط در ناحیه‌های مجاز مثل Stepbar، جدول‌ها، ریل‌های مرحله‌ای و نوارهای انتخابی
- حفظ قفل سراسری صفحه برای جلوگیری از drift افقی در iPhone/Safari
- عدم تغییر Theme، رنگ‌ها، کارت‌ها، Header/Footer یا ساختار بصری V11

## فایل اصلی Patch

```text
src/styles/v11-scroll-access-patch.css
```

## نکته مهم

این نسخه برخلاف V13 بازطراحی سراسری نیست؛ فقط نسخه V11 را با دسترسی بهتر به اسکرول عمودی و افقی آماده می‌کند.
