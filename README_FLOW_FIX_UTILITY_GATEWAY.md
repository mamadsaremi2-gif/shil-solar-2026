# SHIL Flow Fix + Utility Gateway Separation

این آپدیت مسیرهای پروژه جدید، سناریوهای آماده و نیروگاهی را از هم جدا می‌کند.

## اصلاحات اصلی

1. **پروژه جدید معمولی**
   - هنگام ورود به صفحه اطلاعات پروژه، فلگ‌های سناریوی آماده پاک می‌شوند.
   - مسیر عادی به صورت مرحله‌ای ادامه می‌دهد:
     `اطلاعات پروژه → شرایط محیطی → انتخاب مسیر پروژه → روش محاسبات → ورودی‌ها → پیکربندی → چکیده → اجرا`

2. **سناریوهای آماده**
   - فقط وقتی از مسیر `/scenarios` وارد شده باشیم، بعد از شرایط محیطی به لیست تجهیزات سناریو می‌رود.
   - منطق سناریو دیگر روی پروژه جدید معمولی سوار نمی‌شود.

3. **صفحه انتخاب مسیر پروژه**
   - کارت‌های «در حال توسعه» و Future حذف/فیلتر شدند.
   - کارت مستقل «نیروگاهی» اضافه شد.

4. **جداسازی منطق نیروگاهی**
   - منطق نیروگاهی از مسیر «برق خورشیدی با پنل» و «برق اضطراری» جدا شد.
   - تنظیمات Utility/MV/Transformer/Grid Study فقط در مسیر مستقل «نیروگاهی» نمایش داده می‌شود.

5. **هشدار هوشمند توان بالای ۳۰kW**
   - اگر کاربر در مسیر پنل خورشیدی توان بالاتر از ۳۰kW وارد کند، اپ به‌جای باز کردن بلوک‌های نیروگاهی داخل همان صفحه، هشدار می‌دهد و دکمه ورود به درگاه نیروگاهی نمایش می‌دهد.
