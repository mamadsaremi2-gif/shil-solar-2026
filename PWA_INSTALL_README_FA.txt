بسته آماده آیکون PWA شیل

محتویات:
- index.html برای ریشه پروژه
- پوشه public کامل و اصلاح‌شده
- آیکون iOS با اندازه 180x180
- آیکون‌های Android/PWA با اندازه‌های 192x192 و 512x512
- آیکون maskable مجزا برای Android
- favicon های 32 و 48 و ICO
- manifest.json و manifest.webmanifest هماهنگ

روش نصب:
1) فایل index.html را در ریشه پروژه جایگزین کنید.
2) پوشه public را در ریشه پروژه جایگزین/ادغام کنید.
3) npm run build را اجرا کنید.
4) تغییرات را Commit و Push کنید.
5) پس از Deploy، آیکون قبلی را از Home Screen حذف کنید.
6) Safari را کامل ببندید، سایت را دوباره باز کنید و Add to Home Screen را بزنید.

نکته: iOS آیکون Home Screen را شدیداً Cache می‌کند؛ حذف آیکون قبلی الزامی است.
