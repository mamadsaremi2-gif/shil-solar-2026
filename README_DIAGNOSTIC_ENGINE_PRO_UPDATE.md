# SHIL Professional Diagnostic Engine Update

این آپدیت فقط ماهیت مهندسی دارد و هیچ منطق خرید، فروش، قیمت، فروشگاه یا مارکت‌پلیس به پروژه اضافه نمی‌کند.

## فایل‌های اصلی اضافه/تغییر داده‌شده

- `src/core/calculation/solarDiagnosticEngine.js`
- `src/core/calculation/solarAutoDesignEngine.js`
- `src/calculation/diagnostics/diagnosticRules.js`
- `src/pages/project/RunCalculation.jsx`
- `src/styles/app.css`

## قابلیت‌های Diagnostic Engine حرفه‌ای

- امتیاز سلامت طراحی از ۱۰۰
- وضعیت نهایی طراحی: تأیید، نیازمند بازبینی، نیازمند اصلاح، متوقف به دلیل خطای بحرانی
- خطای بحرانی، خطای مهندسی، هشدار و تأیید
- تحلیل انرژی روزانه PV و درصد پوشش مصرف
- تحلیل Undersizing و Oversizing پنل
- تحلیل PSH و تلفات محیطی
- تحلیل توان دائم و Surge اینورتر
- کنترل سقف توان ورودی PV اینورتر
- کنترل محدوده MPPT، Vmp گرم و Voc سرد
- تحلیل ظرفیت باتری و خودکفایی
- تحلیل DoD و نسبت ظرفیت قابل استفاده باتری
- تحلیل جریان DC و حفاظت
- پیشنهاد اصلاح کاملاً فنی و مهندسی

## خروجی UI

در صفحه خروجی نهایی، کارت جدید «Diagnostic Engine حرفه‌ای» اضافه شده که موارد زیر را نمایش می‌دهد:

- امتیاز سلامت طراحی
- تعداد خطاهای بحرانی
- تعداد خطاهای مهندسی
- تعداد هشدارها
- فهرست خطاها/هشدارهای قابل اقدام
- پیشنهاد اصلاح مهندسی برای هر مورد
