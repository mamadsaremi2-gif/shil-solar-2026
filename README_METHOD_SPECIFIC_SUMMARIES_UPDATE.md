# SHIL Method-Specific Summaries Update

این آپدیت چکیده و خروجی نهایی مسیرهای محاسبات را از هم جدا می‌کند، در حالی که همه مسیرها همچنان از موتور محاسباتی یکپارچه استفاده می‌کنند.

## تغییرات اصلی
- اضافه شدن `src/core/summary/methodSummaryEngine.js`
- هر مسیر چکیده اختصاصی خودش را می‌سازد:
  - لیست تجهیزات
  - پروفایل مصرف
  - انرژی روزانه
  - توان کل
  - جریان کل
  - توان پنل خورشیدی
  - برق اضطراری
- `SummaryPage.jsx` از Summary Engine مشترک استفاده می‌کند.
- `RunCalculation.jsx` خروجی نهایی اختصاصی هر مسیر را نمایش می‌دهد.
- مسیرهای بارمحور دیگر خروجی PV/MPPT/تقسیم پنل نمی‌گیرند.
- مسیر `solar_panel_power` همچنان چکیده PV اختصاصی خود را دارد.

## قانون معماری
UI و Summary برای هر روش جدا است؛ اما محاسبات از موتور مشترک خوانده می‌شود.
